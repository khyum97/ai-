# 인공지능학개론 팀과제 보고서 및 실행 가이드

* **팀원:** 박재우, 염지훈, 오형우
* **제출 과제:** student_dropout_prediction.ipynb (학생 중도포기 여부 예측 모델)

본 프로젝트는 이번 학기 인공지능학개론 수업을 통해 처음으로 기계학습(머신러닝)과 인공신경망(딥러닝)을 학습하며 진행한 프로젝트입니다. 대학생들의 배경 데이터와 성적 데이터를 기반으로, 어떤 학생이 학업을 중도에 포기할 가능성이 높은지 학습하고 이를 예측하는 분류 모델을 개발하였습니다.

---

## 1. 소스코드 관련 설명

노트북 소스코드는 수업 시간과 실습 과제에서 배운 내용들을 순서대로 적용하여 작성하였습니다. 전체 코드는 크게 5개 단계로 실행됩니다.

### Section 1. Setup & Data Loading (환경 설정 및 데이터 불러오기)
* 파이썬 데이터 분석에 필수적인 기본 라이브러리들과 머신러닝 라이브러리(Scikit-Learn), 딥러닝 라이브러리(PyTorch)를 가져옵니다.
* 구글 코랩(Colab)이나 로컬 PC 폴더에서 데이터 파일(dataset.csv)을 정상적으로 읽어옵니다.

### Section 2. Data Preprocessing (데이터 전처리)
* **타겟 변수 라벨링:** 데이터셋의 타겟값 중 'Dropout'(중도포기)은 1로 지정하고, 그 외의 상태('Graduate' 및 'Enrolled')는 모두 0으로 묶어 두 가지 분류 상태(0 또는 1)로 변환해 줍니다.
* **범주형 데이터 변환:** 문자 형태로 된 데이터들을 모델이 연산할 수 있도록 수치형 숫자로 바꾸어 줍니다.
* **데이터 정규화 (Scaling):** 각 항목마다 값의 크기 범위(예: 나이와 학비의 숫자 격차)가 너무 달라 학습이 왜곡되는 문제를 막기 위해 StandardScaler를 사용하여 모든 데이터의 스케일을 표준화합니다.
* **학습/테스트 데이터 분할:** 데이터를 모델 학습용(70%)과 평가용(30%)으로 나눕니다. 클래스 비율이 치우치지 않게 stratify 옵션을 적용했습니다.
* **불균형 데이터 해결 (SMOTE):** 중도포기 학생 비율이 상대적으로 적어 학습이 편향되는 문제를 해결하기 위해, 수업에서 배운 SMOTE 기법을 사용하여 훈련 데이터를 복사·합성해 두 클래스의 개수를 동일하게 맞춰 줍니다.

### Section 3. Exploratory Data Analysis (EDA, 데이터 분석 및 시각화)
* SMOTE 처리 후에 데이터 균형이 어떻게 맞춰졌는지 그래프로 시각화하여 확인합니다.
* 피처(특징 변수)들의 전반적인 데이터 분포를 확인합니다.
* 고차원의 데이터 변수들을 2차원으로 축소해 주는 주성분 분석(PCA) 기법을 사용하여, 데이터들이 클래스별로 어떻게 분포해 있는지 그래프로 확인해 봅니다.

### Section 4. Model Development (모델 개발 및 학습)
수업 시간에 배운 다양한 분류 알고리즘을 구현하고 비교합니다.
* **로지스틱 회귀 (Logistic Regression):** 가장 기본적인 선형 분류 모델로 학습을 수행합니다.
* **서포트 벡터 머신 (SVM):** 비선형 경계를 찾아 예측해 주는 SVM 알고리즘을 사용하며, 최적의 하이퍼파라미터(C, gamma) 값을 그리드 서치 기법으로 찾아 적용합니다.
* **랜덤 포레스트 (Random Forest):** 여러 결정 트리를 결합하는 앙상블 기법 모델로 학습시킵니다.
* **인공신경망 (PyTorch MLP):** PyTorch를 이용해 입력층, 은닉층(3개 레이어), ReLU 활성화 함수, 그리고 과적합을 방지하기 위한 Dropout 레이어를 직접 연결한 다층 퍼셉트론 모델을 설계하고 미니배치 학습 루프로 50번 학습을 수행합니다.

### Section 5. Results Analysis (결과 분석 및 평가)
* 실습 과제에서 작성해 보았던 커스텀 Evaluation Metrics 코드를 사용하여 각 모델의 정확도, 정밀도, 재현율, F1-Score를 각각 계산해 봅니다.
* 각 모델이 어떤 예측에서 실수를 많이 했는지 비교하기 위해 혼동 행렬(Confusion Matrix) 격자 그래프를 그려서 확인합니다.

## 2. 실행 방법

실행하기 전에 파일의 실행 환경 및 필수 패키지가 필요합니다.

### 실행 방법 안내
1. 파이썬 및 패키지들이 설치된 터미널(명령 프롬프트)을 실행합니다.
2. 아래 명령어를 입력하여 필요한 라이브러리를 먼저 설치합니다.
   ```bash
   pip install pandas numpy scikit-learn imblearn matplotlib seaborn torch
   ```
3. 노트북 파일(`student_dropout_prediction.ipynb`)과 데이터 파일(`dataset.csv`)을 **같은 폴더**에 둡니다.
4. 아래 명령어로 주피터 노트북을 켭니다.
   ```bash
   jupyter notebook
   ```
5. 주피터 노트북 화면이 열리면 `student_dropout_prediction.ipynb` 파일을 클릭하여 엽니다.
6. 상단 메뉴의 "Cell" -> "Run All"을 클릭해 모든 셀을 처음부터 끝까지 순서대로 실행시킵니다.

## 3. 학습 결과 및 분석 요약

테스트 데이터를 통해 각 모델들을 평가한 실제 성능 결과표입니다.

| 평가 대상 모델 | 정확도 (Accuracy) | 정밀도 (Precision) | 재현율 (Recall) | F1-Score (중도포기 클래스 1) |
| :--- | :---: | :---: | :---: | :---: |
| **SVM (RBF Kernel)** | **87.88%** | **80.09%** | **82.90%** | **0.8147** |
| 로지스틱 회귀 (Logistic Regression) | 87.42% | 77.66% | 85.48% | 0.8138 |
| 다층 인공신경망 (PyTorch MLP) | 87.88% | 81.37% | 80.80% | 0.8108 |
| 랜덤 포레스트 (Random Forest) | 87.58% | 81.19% | 79.86% | 0.8052 |

### 결과 결론
* 학습 및 평가 결과, 본 과제에서는 **SVM (RBF 커널)** 모델이 중도포기 학생을 예측하는 F1-Score 수치가 0.8147로 가장 높게 나타났습니다.
* 수치 위주의 정형 데이터셋에서는 데이터 수가 상대적으로 적기 때문에 복잡한 딥러닝(PyTorch MLP) 모델보다 튜닝된 머신러닝 알고리즘들이 학습 효율과 성능에서 비슷하거나 약간 더 좋은 성능을 보인다는 실습 결과를 얻었습니다.
